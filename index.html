CSS Sizing Guide!

Types of styles

rem takes into account the font size of the root element

em relates to the font size of the element

vw is relative to 1% of the width of the viewport

vh is realtive to 1% of the height of the viewport

px 1/96 of an inch, however the pixels also depend on the device screen size

See the Pen Untitled by aspenrosee (@aspenrosee) on CodePen.

When should you use each one?

rem: This is great for absolute measurement styles, and is realtive to what a device's default size is.

em: This is good for making indents, margins, and measuring the width of an element on a page.

vw/vh: These types of measurements help to make images or text be the full size of the viewport, or you can specify what portion you want it to be.

px: Pixels are onyl used for elements that should stay the same size no matter what context it is in. Pixels generally should not be used otherwise, as they are not as dynamic as other options.
